Publishing checklist

  • Export your pack as a single .zip.
  • Overwrite mc/v/latest/modpack.zip with the new file so new-arcadia.net/mc/v/latest stays current.
  • (Optional) Archive the same file inside mc/v/releases/<version>/ for historical downloads.

Version history

Maintain a changelog or extra download links in the version archive so people can roll back if needed.
