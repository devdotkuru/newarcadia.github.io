How versioning works

Each release lives in mc/v/<version> with the exported pack archive. The latest folder automatically points people to the newest version so you can share new-arcadia.net/mc/v/latest everywhere.

  • Copy your exported modpack (e.g. NewArcadia-1.2.0.zip) to mc/v/releases/1.2.0/.
  • Update mc/v/latest/latest.json so the downloadUrl points to the new archive.
  • Commit the changes and push to publish the update instantly.

Automation ready

You can also script the update process. Replace the file in the releases folder, bump the JSON entry, and your CI can handle the rest.
