Versioned modpack downloads

Keep a simple changelog here so players can download specific builds. Copy each exported pack into mc/v/releases/<version>/ and link to it below. The latest link stays pointed at mc/v/latest/modpack.zip, so you only need to overwrite that one file for fresh releases.
