## Project setup

1. Clone the repository and ensure you have the official Halo Wars files.
2. Install **Haruspis** and the Asset Bundler.
3. Launch the tools from `docs/tools/` for guided walkthroughs.

## Configuration tips

- Keep your mods under `New Arcadia/mods/`.
- Use semantically meaningful version numbers (0.1.0, 0.2.0).
- Refer to the engine tables for Granny/WWise compatibility before adjusting assets.
